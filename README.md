# log4v

  [![Apache 2.0 License](https://img.shields.io/badge/license-Apache_2.0-green.svg?style=flat)](./LICENSE)

Logging framework for V.


## Setup and run

Get the module via `vpm`:
```bash
v install log4v
```
then use by importing the module 'log4v'.
